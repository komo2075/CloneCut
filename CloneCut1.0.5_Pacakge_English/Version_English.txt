CloneCut – Version Information

Version: v1.0.5
Release Date: 2026-03-12

Description

CloneCut v1.0.5 is a stability and bug-fix update based on the initial public release.
This version focuses on improving the reliability of the naming block system and fixing several workflow issues. No new major features were added, but multiple bugs and edge cases were resolved to provide a smoother user experience.

CloneCut remains a lightweight local tool designed to batch resize vertical 9:16 videos into multiple aspect ratios such as 1:1, 16:9, 4:5, and 2:3, with a modular naming block system for customizable file naming.

Improvements and Fixes
Naming Block System

Fixed an issue where the Delete button did not work for blocks other than the default size block in Module 3.

Fixed an issue where validation prompts opened as independent pages, causing the naming queue to be lost when returning to the interface.

Fixed an issue where incomplete variant names would automatically insert size numbers. Empty fields are now skipped correctly.

Preset and Template System

Fixed an issue where variant names could not be saved to Module 4 presets.

Fixed an issue where selecting a variant block in the Module 3 queue prevented viewing its details.

Fixed an issue where variant names were incorrectly detected as incomplete during download even when correctly filled.

Workflow Behavior

Fixed an issue where variant presets from Module 4 could not be correctly added to the Module 3 naming queue.

Fixed a synchronization issue where temporary edits to templates in Module 3 affected other template blocks created from the same preset.

Fixed an issue where newly added template blocks were affected by earlier temporary edits.

Temporary variant names created in Module 3 can now be correctly saved when clicking Save Template.
