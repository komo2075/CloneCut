# CloneCut
A lightweight local tool designed for **game user acquisition (UA) video production workflows**.
CloneCut converts a **single 9:16 source video** into multiple aspect ratios and automatically generates structured filenames using a modular naming system.
All exported videos are packaged into a ZIP file for easy download.
Built for **UA creators, editors, and production teams** who need to quickly generate multiple ad formats.

---
## ✨ Features
- Batch export multiple video aspect ratios from one source video  
- Modular **Naming Block System** for customizable filenames  
- Automatic **ZIP packaging** of exported videos  
- Built-in **FFmpeg backend**, no external configuration required  
- Fully **offline workflow**, no cloud upload  

---
## 📐 Supported Output Formats
CloneCut supports the following aspect ratios:
- **9:16** — 1080 × 1920  
- **1:1** — 1080 × 1080  
- **16:9** — 1920 × 1080  
- **2:3** — 1080 × 1620  
- **4:5** — 1080 × 1350  
All exported videos are generated at **60 fps**.

---
## ⚙️ How It Works
1. Upload a **9:16 source video**  
2. Select the output aspect ratios  
3. Configure filenames using the **Naming Block System**  
4. Set the ZIP package name  
5. Click **Generate and Download**
CloneCut automatically exports all selected versions and packages them into a ZIP file.

---
## ⏱ Processing Time
Processing time depends on the **length of the video** and **computer hardware performance**.
For a typical **30–60 second video**, CloneCut usually generates all outputs within **1–3 minutes**.
Actual processing time may vary depending on system configuration.
