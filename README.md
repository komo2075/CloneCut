CloneCut

CloneCut 是一个轻量级本地工具，专为 游戏买量（Game UA）视频制作流程设计。

CloneCut is a lightweight local tool designed for game user acquisition (UA) video production workflows.

它可以将 一个 9:16 竖版源视频批量转换为多个视频比例，并自动生成文件名并打包下载。

It converts a single 9:16 source video into multiple aspect ratios while automatically generating filenames and exporting them as a ZIP package.

Features | 功能特点

Batch export multiple video aspect ratios
批量导出多个视频比例

Modular Naming Block System for flexible file naming
模块化 命名积木系统 自由组合文件名

Automatic ZIP packaging of exported videos
自动打包导出视频为 ZIP 文件

Built-in FFmpeg backend
内置 FFmpeg 后端

Fully offline workflow
完全离线运行

Supported Output Formats | 支持导出比例

9:16 — 1080 × 1920

1:1 — 1080 × 1080

16:9 — 1920 × 1080

2:3 — 1080 × 1620

4:5 — 1080 × 1350

All exported videos are generated at 60 fps.
所有导出视频统一为 60 fps。

How It Works | 使用流程

Upload a 9:16 source video
上传 9:16 竖版源视频

Select output aspect ratios
选择需要导出的比例

Configure filenames using the Naming Block System
使用 命名积木系统设置文件名

Set the ZIP package name
设置 ZIP 文件名称

Click Generate and Download
点击 Generate and Download

Processing Time | 生成时间

Processing time depends on video length and hardware performance.

For a typical 30–60 second video, CloneCut usually completes the export within 1–3 minutes.

生成时间取决于视频时长和电脑硬件性能。

对于 30–60 秒视频，通常 1–3 分钟即可完成全部导出。
