CloneCut

CloneCut is a lightweight local tool designed for game user acquisition video production workflows.

It allows users to quickly convert a single 9:16 source video into multiple aspect ratios such as 1:1, 16:9, 4:5, and 2:3, while automatically generating structured filenames and exporting all versions as a ZIP package.

CloneCut is built to simplify repetitive export and naming tasks commonly found in UA video production.

Features
-Batch export multiple video aspect ratios from one source video
-Modular Naming Block System for customizable file naming
-Automatic ZIP packaging of all exported videos
-Built-in FFmpeg backend, no external configuration required
-Fully offline operation, no data upload or cloud processing

Supported Output Formats
CloneCut supports the following aspect ratios:

9:16 — 1080 × 1920
1:1 — 1080 × 1080
16:9 — 1920 × 1080
2:3 — 1080 × 1620
4:5 — 1080 × 1350
All exported videos are generated at 60 fps.

How It Works
1.Upload a 9:16 source video
2.Select the output aspect ratios
3.Configure filenames using the Naming Block System
4.Set the ZIP package name
5.Click Generate and Download
CloneCut will automatically export all selected versions and package them into a ZIP file.

Processing Time
-Processing time depends on the length of the source video and the hardware performance.
-For a typical 30–60 second video, CloneCut usually generates all outputs within 1–3 minutes.
-Actual processing time may vary depending on computer configuration.

Requirements
-Windows 10 or later
-Modern browser (Chrome / Edge / Firefox)
No installation required.
