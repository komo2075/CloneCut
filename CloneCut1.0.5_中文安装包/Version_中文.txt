CloneCut – 版本信息
版本号： v1.0.5
发布日期： 2026-03-12

简介
CloneCut v1.0.5 是基于首个公开版本发布后的 稳定性与问题修复更新。
本版本主要针对命名积木系统及工作流程中的多个问题进行修复与优化。没有新增主要功能，但通过修复若干 bug 和边界情况，提升了整体使用稳定性与体验。
CloneCut 仍然是一个轻量级本地工具，用于将 9:16 竖版视频批量转换为多种比例（包括 1:1、16:9、4:5、2:3 等）。
通过模块化命名积木系统，用户可以灵活组合文件命名规则，并自动批量导出高质量视频。

本版本更新：
命名积木系统
-修复 Module 3 中 除默认尺寸积木外删除按钮失效的问题
-修复检测提示会以独立页面形式打开，返回后导致命名队列丢失的问题
-修复区别名未填写完整时 自动添加尺寸编号的问题，现在空内容会正确跳过

预设与模板系统
-修复 区别名无法保存为 Module 4 预设的问题
-修复在 Module 3 队列中选择区别名后 无法查看其详情的问题
-修复即使区别名填写正确，下载时仍被检测为未填写并被替换为尺寸编号的问题

工作流程优化
-修复 Module 4 的区别名预设 无法正确添加到 Module 3 命名队列的问题
-修复 Module 3 中 临时修改模板会同步影响同源模板块的问题
-修复新增模板块会受到之前临时修改影响的问题
-Module 3 中设置的 临时区别名现在可以正确保存到模板

系统要求：
Windows 10 或更高版本

浏览器支持
Chrome / Edge / Firefox
